# Eliminacion de Recurso dentro de Azure RM

## Introducción

Este script permite la creacion de un reporte de los Recursos en Azure mediante powershell.

## Ejecución

###### Powershell

`.\PS_ARM_ResourceReport.ps1`

## Help

Para tener mas informacion sobre el contenido del script se puede ver su informcion mediante:

###### Powershell

`Get-Help .\PS_ARM_ResourceReport.ps1`
